# Densità ristoranti-francia
## Cosa fa questo progetto
Questo progetto crea una mappa che mostra la densità dei ristoranti nelle regioni del sud della Francia.

## Come si avvia
1. Clona questo repository
2. Apri RStudio
3. Apri il file mappa–ristoranti.R
4. Esegui tutto il codice

## Dati utilizzati 
- Ristoranti: dataset CSV da R Graph Gallery
- Confini regioni: GeoJSON da Gregorie David su GitHub

## Requisiti
- R versione 4.0
- Pacchetti R: ggplot2, sf, dplyer, readr

## Limiti noti
- I dati potrebbero non essere aggiornati
- Alcune regioni potrebbero avere dati mancanti
